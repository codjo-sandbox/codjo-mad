Package permettant de créer facilement la barre des menus avec la toolbar associé.

Utilisation du package

Lecture du fichier de paramètrage XML. 
    JMenuBar menuBar =
        MenuFactory.buildMenuBar(new InputSource(maclass.getResourceAsStream("/conf/menu.xml"))
            , menuCtxt);

Extension

Il est possible de fournir un menu deja construit : 
        menuCtxt.putProperty("Menu Fenetre", new WindowMenu(desktop));
        ...MenuFactory.buildMenuBar...
